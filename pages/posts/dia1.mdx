# Primeiros passos

    Este projeto será criado usando nextJS para front e backend em nodeJS e firestore para armazanamento e persistencia de dados. Naturalmente o primeiro passo é inicializar o Next app com o comando Yarn

'''
yarn create next-app
'''

## criação do projeto na firebase

    O primeiro passo é criar um novo projeto da Firebase e um novo aplicativo Web na plataforma. AO finalizar o processo as chaves para utilização dos recursos Firebase serão disponibilizadas pela Google. Por questões de segurança aloquei essas informações em um arquivo .env.local, que posteriormente será acessado pelo Next atraves do metodo process.env

    Com o projeto criado podemos agora ativar a instancia de Firestore em tempo real.
